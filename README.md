# Online-shop
It is a simple online store with html and css frontend and Django backend <br>

1. Install these to run <br>
  ```
  pip install Django
  ```
```
pip install Pillow
```
2. Run the Server:
   - Execute the following command:
     ```
     python manage.py runserver
     ```
     
3. Use the project :
   - Open your browser and enter this url
     ```
     127.0.0.1:800
     ```
